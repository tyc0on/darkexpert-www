# Site Positioning

## One-Liner
DarkExpert is a personal expert agent that turns messy human intent into durable, harness-ready work for the dark factory.

## Audience
- Technical founders
- Product-minded operators
- Small teams that want leverage without hiring a large engineering org

## Messaging Requirements
- Lead with the operating model, not generic AI claims.
- Explain that humans steer and agents do the engineering.
- Emphasize durable outputs: plans, docs, code, tests, and reviewable artifacts.
- Make the system feel disciplined, not magical.

## Desired Visitor Reaction
"This feels like an actual operating system for software execution, not just a chat UI."
